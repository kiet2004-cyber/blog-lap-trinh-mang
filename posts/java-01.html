Java là gì? Cài đặt và chương trình đầu tiên

Java là ngôn ngữ lập trình hướng đối tượng, đa nền tảng, được sử dụng rộng rãi trong phát triển phần mềm, web và ứng dụng doanh nghiệp.

1. Đặc điểm của Java

  • Hướng đối tượng
  • Chạy trên nhiều hệ điều hành
  • Bảo mật cao

2. Cài đặt Java

Tải JDK từ trang chủ Oracle hoặc OpenJDK, sau đó cài đặt theo hướng dẫn.

3. Chương trình Java đầu tiên

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

Chương trình trên in ra dòng chữ Hello Java.
